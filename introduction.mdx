# Welcome to KMUTNB Mobile App

A Flutter mobile application created for King Mongkut's University of Technology North Bangkok (KMUTNB). This app provides seamless access to the university website and an interactive campus map, making it easier for students and visitors to navigate and access university resources.

## What is KMUTNB Mobile App?

The KMUTNB Mobile App is a cross-platform mobile application that combines two essential features:

- **Direct access** to the official KMUTNB website through an integrated WebView
- **Interactive campus map** powered by Google Maps, showing the Faculty of Technology and Management location

Built with Flutter, the app delivers a native experience on both Android and iOS platforms with a modern Material Design 3 interface.

## Key features

The KMUTNB Mobile App includes the following capabilities:

### University website access

Built-in WebView that loads the official KMUTNB website (https://kmutnb.ac.th/) with full browsing capabilities. Access all university resources directly from the app without switching to an external browser.

### Campus map

Interactive Google Maps integration showing the Faculty of Technology and Management location. Use the floating action button to quickly recenter the map on the faculty location.

### User-friendly navigation

Simple bottom navigation bar for easy switching between the website and map features. The intuitive interface ensures smooth navigation throughout the app.

### Material Design 3

Modern UI with Material 3 components and theming, providing a contemporary look and feel that follows the latest design guidelines.

## Screenshots
