# Installation

Follow these steps to install and set up the KMUTNB Mobile App on your development machine.

## Prerequisites

Before you begin, ensure you have the following installed:

<Steps>
  <Step title="Flutter SDK 3.7.0 or higher">
    Download and install the Flutter SDK from the [official Flutter website](https://flutter.dev/docs/get-started/install). Verify your installation by running:
    
    ```bash
    flutter --version
    ```
  </Step>
  
  <Step title="Dart SDK 3.7.0 or higher">
    The Dart SDK is included with Flutter. The app requires Dart SDK 3.6.0 or higher (as specified in `pubspec.yaml`).
  </Step>
  
  <Step title="Platform-specific SDKs">
    Depending on your target platform:
    - **Android**: Install Android SDK and Android Studio
    - **iOS**: Install Xcode (macOS only)
  </Step>
</Steps>

## Install the app

<Steps>
  <Step title="Clone the repository">
    Clone the KMUTNB App repository to your local machine:
    
    ```bash
    git clone https://github.com/yourusername/kmutnb-app.git
    ```
  </Step>
  
  <Step title="Navigate to the project directory">
    Change to the project directory:
    
    ```bash
    cd kmutnb-app
    ```
  </Step>
  
  <Step title="Install dependencies">
    Install all required Flutter packages:
    
    ```bash
    flutter pub get
    ```
    
    This will install the following dependencies:
    - `google_maps_flutter: ^2.10.1` - Google Maps integration
    - `geolocator: ^13.0.2` - Location services
    - `webview_flutter: ^4.10.0` - WebView functionality
    - `http: ^1.3.0` - Network requests
    - `flutter_lints: ^5.0.0` - Code quality linting
  </Step>
  
  <Step title="Verify installation">
    Check that Flutter can detect your connected devices:
    
    ```bash
    flutter devices
    ```
  </Step>
</Steps>

<Note>
  The app requires a Google Maps API key to display the campus map. You'll need to configure this before running the app. See the [Google Maps configuration](/configuration/google-maps) page for detailed instructions.
</Note>
