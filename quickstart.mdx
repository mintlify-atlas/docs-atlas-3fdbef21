# Quickstart

Get started with the KMUTNB Mobile App quickly by following these steps. You'll have the app running on your device or emulator in no time.

## Prerequisites

Before you begin, make sure you have:
- Flutter SDK 3.7.0+ installed
- Dart SDK 3.6.0+ installed
- Android Studio or Xcode (depending on your target platform)
- A Google Maps API key ([get one here](https://console.cloud.google.com/))

## Get started

<Steps>
  <Step title="Clone the repository">
    Clone the KMUTNB Mobile App repository to your local machine:
    
    ```bash
    git clone https://github.com/jezlol/Mobile_final_f1.git
    cd Mobile_final_f1
    ```
  </Step>
  
  <Step title="Install dependencies">
    Install all required Flutter packages:
    
    ```bash
    flutter pub get
    ```
    
    This will install Google Maps Flutter, WebView Flutter, Geolocator, and other dependencies.
  </Step>
  
  <Step title="Configure Google Maps API key">
    Add your Google Maps API key to the platform-specific configuration files:
    
    **For Android** (`android/app/src/main/AndroidManifest.xml`):
    ```xml
    <meta-data
      android:name="com.google.android.geo.API_KEY"
      android:value="YOUR_API_KEY_HERE"/>
    ```
    
    **For iOS** (`ios/Runner/AppDelegate.swift`):
    ```swift
    GMSServices.provideAPIKey("YOUR_API_KEY_HERE")
    ```
    
    See the [Google Maps configuration](/configuration/google-maps) guide for detailed instructions.
  </Step>
  
  <Step title="Connect a device or start an emulator">
    Connect your Android or iOS device, or start an emulator:
    
    ```bash
    flutter devices
    ```
    
    This command will show all available devices.
  </Step>
  
  <Step title="Run the app">
    Launch the app on your connected device or emulator:
    
    ```bash
    flutter run
    ```
    
    The app will compile and launch on your device.
  </Step>
</Steps>

## What you'll see

Once the app launches, you'll see two main screens accessible via bottom navigation:

1. **Home tab**: Displays the KMUTNB university website (https://kmutnb.ac.th/) in an embedded WebView
2. **Map tab**: Shows an interactive Google Maps view centered on the Faculty of Technology and Management with a marker

## Understanding the app structure

The KMUTNB Mobile App is built with a simple, clean architecture:

```dart
// Main app structure from lib/main.dart
void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      title: 'KMUTNB App',
      theme: ThemeData(
        colorScheme: ColorScheme.fromSeed(seedColor: Colors.orange),
        useMaterial3: true,
      ),
      home: const MainScreen(),
    );
  }
}
```

The app uses:
- **Material Design 3** with an orange color scheme
- **IndexedStack** to preserve screen state when switching tabs
- **Bottom navigation** for easy switching between WebView and Map screens

## Project files

Key files in the project:

- `lib/main.dart` - App entry point and navigation setup
- `lib/screens/webview_screen.dart` - WebView implementation
- `lib/screens/map_screen.dart` - Google Maps implementation
- `pubspec.yaml` - Dependencies and project configuration
