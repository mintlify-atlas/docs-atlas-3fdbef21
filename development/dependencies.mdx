## Core dependencies

These dependencies are essential for the app's main features:

### google_maps_flutter

<ParamField path="google_maps_flutter" type="package" default="^2.10.1">
  **Purpose**: Google Maps integration
  
  Provides the `GoogleMap` widget and controller for displaying interactive maps in the app. Used in `MapScreen` to show the Faculty of Technology and Management location with markers and camera controls.
  
  **Key features used**:
  - GoogleMap widget
  - Marker placement
  - Camera positioning and animation
  - Map type configuration
</ParamField>

### geolocator

<ParamField path="geolocator" type="package" default="^13.0.2">
  **Purpose**: Location services
  
  Provides cross-platform (iOS, Android) geolocation capabilities including location permissions, current position, and distance calculations. While included in dependencies, this package is available for future location-based features.
  
  **Potential use cases**:
  - Getting user's current location
  - Calculating distance to campus
  - Location-based notifications
</ParamField>

### webview_flutter

<ParamField path="webview_flutter" type="package" default="^4.10.0">
  **Purpose**: WebView functionality
  
  Enables embedding web content directly in the Flutter app. Used in `WebViewScreen` to display the KMUTNB website (https://kmutnb.ac.th/) with full JavaScript support and navigation capabilities.
  
  **Key features used**:
  - WebViewController for page control
  - JavaScript mode configuration
  - NavigationDelegate for event handling
  - Page loading callbacks
</ParamField>

### http

<ParamField path="http" type="package" default="^1.3.0">
  **Purpose**: Network requests
  
  Provides a composable API for making HTTP requests. While included in dependencies, this package is available for future API integration and data fetching needs.
  
  **Potential use cases**:
  - Fetching campus news and announcements
  - API integration with university systems
  - Remote data synchronization
</ParamField>

### cupertino_icons

<ParamField path="cupertino_icons" type="package" default="^1.0.8">
  **Purpose**: iOS-style icons
  
  Provides iOS-style icons for use with the CupertinoIcons class. Ensures consistent icon appearance across platforms.
</ParamField>

## Installing dependencies

After cloning the project or modifying `pubspec.yaml`, run:

```bash
flutter pub get
```

This command downloads all dependencies and makes them available to your project.

## Updating dependencies

To check for newer versions of dependencies:

```bash
flutter pub outdated
```

To upgrade dependencies to the latest compatible versions:

```bash
flutter pub upgrade
```

<Info>
For major version upgrades that might include breaking changes, use `flutter pub upgrade --major-versions` and carefully test your app afterwards.
</Info>
